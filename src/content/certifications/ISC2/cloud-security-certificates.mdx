## Overview

The **ISC2 Cloud Security Certificates (CSC)** are designed to help cybersecurity professionals advance their skills in
cloud security. These certificates provide a pathway to the CCSP certification and demonstrate a commitment to
maintaining ISC2 certifications through professional development.

The CSC validates an individual's ability to:

- Understand cloud computing concepts and architecture.
- Apply security principles to cloud environments.
- Implement secure cloud migration strategies.

## Course Details

The CSC bundle consists of three certificates:

1. **Cloud Basics Certificate**: Covers foundational cloud computing concepts, including drivers for cloud adoption,
   cloud architecture, and service models.
2. **Moving to the Cloud Certificate**: Focuses on the strategic and secure approach to cloud migration using a
   four-phased process.
3. **Working in the Cloud Certificate**: Immerses learners in real-world cloud security challenges from the perspective
   of a cloud security professional.

### Topics Covered:

- Cloud computing fundamentals.
- Cloud security principles and best practices.
- Secure cloud migration planning and execution.
- Scenario-based learning experiences.

## Exam Structure

To earn each CSC, candidates must score 70% or higher on the final assessment. The assessments can be taken directly if
the learner already possesses the required knowledge.

### Key Exam Requirements:

- Passing score of 70% or higher on the final assessment.
- Completion of the learning and assessment or assessment only option.

## Who Should Take CSC?

The CSC is ideal for:

- Cloud Security Specialists looking to enhance their skills.
- Cybersecurity Managers aiming to lead cloud security initiatives.
- Security Operations Specialists focused on secure cloud operations.
