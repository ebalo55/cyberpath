## Overview

The **ISC2 Cloud Security Skill-Builders (CSB)** are designed to help cybersecurity professionals stay current with the
latest trends and best practices in cloud security. These on-demand learning modules provide quick, targeted skill
development to enhance your ability to secure data and critical assets in cloud environments.

The CSB validates an individual's ability to:

- Understand cloud-native security concepts.
- Implement Zero Trust architecture in the cloud.
- Stay at the forefront of emerging cloud security risks and mitigation strategies.

## Course Details

The CSB bundle includes two key modules:

1. **An Orientation to Cloud Native**: This 30-minute video explores the differences between intrusion detection
   system (IDS) services in cloud-native environments and traditional data center IDS appliances. Learners gain the
   knowledge needed to harness the power of cloud-native approaches for scalable, cost-effective intrusion detection.

2. **Zero Trust Architecture in the Cloud**: This 15-minute module combines an article and a skill application exercise
   to teach how to develop a Zero Trust cloud security architecture. It explores cloud-native tools and the concepts of
   a Zero Trust methodology using real-world examples, providing guidance for creating Zero Trust architecture within
   your organization.

### Key Benefits:

- Stay current with emerging cloud security topics.
- Receive quick, on-demand learning that fits your busy schedule.
- Earn CPE credits for maintaining your ISC2 certifications.

## Exam Structure

The CSB modules do not have a formal exam structure. Learners complete the modules at their own pace and earn CPE
credits upon successful completion.

## Who Should Take CSB?

The CSB is ideal for:

- Cloud Security Specialists looking to enhance their skills.
- Cybersecurity Managers aiming to lead cloud security initiatives.
- Security Operations Specialists focused on secure cloud operations.
