## Overview

The **ISC2 Software Security Skill-Builders (SSSB)** are designed to enhance your skills in securing the software
development lifecycle. Created by industry experts, these on-demand courses provide short-format learning tailored to
fit your busy schedule, enabling you to apply best practices from design to deployment.

The certification validates an individualâ€™s ability to:

- Implement security measures throughout the software development lifecycle.
- Secure application programming interfaces (APIs) against vulnerabilities.
- Employ effective testing and monitoring strategies for software security.

## Course Details

The SSSB bundle includes several key modules:

1. **The Future of API Security**
    - **Delivery Method**: On-Demand
    - **Duration**: 15 Minutes
    - **CPE**: 0.50
    - **Proficiency Level**: Foundational
    - This course examines best practices for securing APIs, including penetration testing, secure coding, and regular
      security audits.

2. **Software Security: Fixing Forward at Scale in Real Life**
    - **Delivery Method**: On-Demand
    - **Duration**: 30 Minutes
    - **CPE**: 0.50
    - **Proficiency Level**: Foundational
    - Learn how testing, tools, and human ingenuity can work together to safeguard systems and data from malicious
      attacks.

3. **Application Programming Interface (API) Security Best Practices**
    - **Delivery Method**: On-Demand
    - **Duration**: 15 Minutes
    - **CPE**: 0.50
    - **Proficiency Level**: Intermediate
    - This module covers best practices for API security, including secure protocols, authentication, access controls,
      and monitoring API activity.

### Key Benefits:

- Stay current with emerging software security topics.
- Quick, on-demand learning that fits your busy schedule.
- Earn CPE credits for maintaining your ISC2 certifications.

## Exam Structure

The SSSB modules do not have a formal exam structure. Participants complete the modules at their own pace and earn CPE
credits upon successful completion.

## Who Should Take SSSB?

The SSSB is ideal for:

- Cybersecurity managers looking to enhance their software security skills.
- Risk management specialists aiming to implement effective security measures.
- Security operations specialists focused on securing applications and APIs.
