## Overview

The **Certified AWS Cloud Red Team Specialist (CARTS)** certification is tailored for cybersecurity professionals who
wish to specialize in red teaming within Amazon Web Services (AWS) environments. This certification equips candidates
with the necessary skills to conduct effective penetration tests and security assessments using AWS tools and
methodologies.

The certification validates an individualâ€™s ability to:

- Execute red team operations specifically in AWS cloud environments.
- Identify and exploit vulnerabilities across various AWS services.
- Understand the functionalities of AWS services and how they can be leveraged for red teaming.

## Course Details

The CARTS certification is structured into three comprehensive sections, each focusing on critical aspects of red
teaming in AWS.

### Topics Covered:

- **Introduction to AWS Cloud**:
    - Overview of AWS architecture and security features.
    - Understanding IAM roles, policies, and best practices.

- **Red Team Operations in AWS Cloud Environment**:
    - Techniques for conducting assessments within AWS.
    - Strategies for gaining initial access and performing reconnaissance.

- **AWS Services Functionalities & Exploitation**:
    - In-depth exploration of key AWS services (e.g., EC2, S3, Lambda) and their security implications.
    - Methods for exploiting misconfigurations and vulnerabilities in AWS services.

### Cyber Range Lab Highlights:

Participants will have access to cyber range labs that simulate real-world scenarios, allowing candidates to practice
their skills in a controlled environment.

## Exam Structure

The CARTS certification is awarded upon successful completion of a practical exam that assesses the candidate's ability
to perform red team operations within AWS environments.

### Key Exam Requirements:

- Completion of all course modules.
- Participation in hands-on labs that simulate real-world cloud environments.

## Who Should Take CARTS?

The CARTS certification is ideal for:

- Intermediate to advanced cybersecurity professionals looking to enhance their skills in cloud security.
- Red teamers wanting to specialize in operations within AWS environments.
- Penetration testers seeking to understand the unique challenges posed by cloud infrastructures.
