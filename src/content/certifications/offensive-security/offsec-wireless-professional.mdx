## Overview

The **OffSec Wireless Professional (OSWP)** certification is designed to validate an individual's ability to identify
and exploit vulnerabilities in 802.11 networks. Offered by Offensive Security, this certification is obtained through
the **Foundational Wireless Network Attacks (PEN-210)** course.

The certification demonstrates a learnerâ€™s ability to:

- Identify and exploit common vulnerabilities in wireless networks.
- Conduct wireless security assessments.
- Apply practical skills in a hands-on, self-paced environment.

## Course Details

The OSWP certification is tied to the **PEN-210** course, which introduces the foundations of wireless network security
and explores common vulnerabilities and exploitation techniques.

### Topics Covered:

- Wireless network security fundamentals.
- Common vulnerabilities in 802.11 networks.
- Exploitation techniques for wireless networks.
- Practical skills for wireless security assessments.

## Exam Structure

The OSWP certification is awarded upon successful completion of the online training course and passing the accompanying
exam.

### Key Exam Requirements:

- Completion of the PEN-210 course.
- Passing the certification exam.

## Who Should Take OSWP?

The OSWP is ideal for:

- Aspiring wireless security specialists.
- Penetration testers looking to specialize in wireless security.
- Security professionals aiming to enhance their wireless security skills.
