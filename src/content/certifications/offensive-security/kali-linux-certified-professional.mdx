## Overview

The **Kali Linux Certified Professional (KLCP)** certification is designed for cybersecurity professionals who wish to
validate their skills in using Kali Linux, the industry-standard toolkit for penetration testing. This certification
demonstrates a comprehensive understanding of Kali Linux and its applications in various security roles.

The certification validates an individualâ€™s ability to:

- Install and configure Kali Linux.
- Utilize advanced features and tools for penetration testing.
- Implement security best practices and monitoring techniques.

## Course Details

The KLCP certification is tied to the **PEN-103** course, known as **"Kali Linux Revealed"**. This course provides a
hands-on introduction to the Kali Linux distribution, covering everything from basic usage to advanced security
configurations.

### Topics Covered:

- Introduction to Linux and Kali Linux features.
- Installation methods and troubleshooting.
- Configuration and customization of the Kali environment.
- Securing and monitoring Kali Linux systems.

## Exam Structure

The KLCP certification is awarded upon successful completion of a proctored exam that tests theoretical knowledge
through multiple-choice questions. The exam assesses the understanding of Kali Linux concepts and its practical
applications.

### Key Exam Requirements:

- 80 multiple-choice questions to be completed within 90 minutes.
- Closed-book format with no external resources allowed.

## Who Should Take KLCP?

The KLCP is ideal for:

- Aspiring cybersecurity professionals seeking foundational knowledge in Linux.
- System administrators looking to enhance their security skills.
- Penetration testers wanting to master the use of Kali Linux.
