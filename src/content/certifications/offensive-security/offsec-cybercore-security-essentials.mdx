# Offensive Security CyberCore Certification (OSCC)

## Overview

The **Offensive Security CyberCore Certification (OSCC)** is designed to equip learners with the essential knowledge and
develop the hands-on skills needed to launch a successful career in cybersecurity. The certification focuses on
foundational security concepts and prepares individuals for advanced cybersecurity training.

The certification validates an individualâ€™s ability to:

- Understand core security concepts.
- Develop hands-on cybersecurity skills.
- Use practical techniques to protect systems and networks.

## Course Details

The **OSCC** certification is tied to the **SEC-100** course, known as **"CyberCore - Security Essentials."** This
course serves as a foundational entry into the cybersecurity field and emphasizes hands-on skills combined with
real-world security knowledge.

### Topics Covered:

- Core cybersecurity principles and practices.
- Hands-on skills in system security.
- Network security essentials.
- Incident response fundamentals.

## Exam Structure

To achieve the **OSCC** certification, candidates must complete a comprehensive exam that tests their understanding of
cybersecurity concepts and their ability to apply skills in real-world scenarios.

### Key Exam Requirements:

- A strong understanding of core security concepts.
- Practical application of security skills in a controlled environment.

## Who Should Take OSCC?

The **OSCC** certification is ideal for:

- Entry-level cybersecurity professionals.
- IT specialists looking to build a foundation in security.
- Individuals seeking to transition into the cybersecurity field.
