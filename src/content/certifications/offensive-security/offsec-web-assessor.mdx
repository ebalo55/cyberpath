## Overview

The **Foundational Web Application Assessments with Kali Linux (OSWA)** course by Offensive Security introduces web
application security testing methodology, tools, and techniques in a hands-on, self-paced environment. Completing the
course and passing the exam earns the **OffSec Web Assessor (OSWA)** certification, demonstrating your ability to
identify and exploit vulnerabilities in web applications.

## Course Details

The WEB-200 course emphasizes practical skills through hands-on labs and exercises, providing a deep understanding of
common web vulnerabilities and how to exploit them responsibly.

### Topics Covered:

- Web application security testing methodology.
- Tools and techniques for identifying and exploiting web vulnerabilities.
- Hands-on labs with various web vulnerability scenarios.

## Exam Structure

The OSWA certification is awarded upon successful completion of an exhaustive exam that tests your ability to identify
and exploit web vulnerabilities.

### Key Exam Requirements:

- Comprehensive understanding of web vulnerabilities.
- Practical exploitation skills demonstrated in a controlled environment.

## Who Should Take OSWA?

The OSWA is ideal for:

- Aspiring web application security testers.
- Security professionals looking to enhance their web security skills.
- IT administrators who want to understand web security from an attacker's perspective.
